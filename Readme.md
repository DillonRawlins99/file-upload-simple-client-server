# About
Simple file upload using Node.js back-end, vanilla javascript front-end. No fancy libraries or forms. 

# Environment
OS: Windows 10
Browser: Chrome 98.0.4758.102 
Node: v16.14.0

# Running This Sample
1. Clone and cd into root directory.
2. `npm start`
3. Navigate to http://localhost:8080
